Der Die Das is a fun German article quiz game in which the player tries to guess the correct article for a set of nouns.

Presently the game is in the early development stage:

  • Contains 350 nouns adhering to the Goethe-Institut A1 examination.
  • Words are pronounced uses speech synthesis.
  • Each correct answer earns one point. A false answer resets the score to zero.
  • Various tips to recognize articles for certain nouns are included.
  • Progress and difficult nouns can be viewed from a list overview.
Development roadmap:
  • Add nouns adhering to Goethe-Institut A2 and B1. B2 will be considered if there is sufficient interest.
  • Include more tips.
  • User Interface tweeks.
  • Release test version on Android and iOS.
NOTE: Der Die Das is a game in which the player tests their knowledge of German articles for a set of nouns, it isn't designed to teach vocabulary.
There are presently no plans to include English (or other language) translations for the German nouns.
